What is WebVTT?

Open standard for timed captions.

Why do we need it?
  • Translation of international media
  • Accessibility for impaired hearing or vision
What else can we do with it?
  • Media navigation using Chapter Markers
  • Caption live-streaming video
Why not use a different caption format?
  • Style outputted text by applying CSS to ids and classes
  • Encoded in UTF-8 to support additional languages and character sets
